# Menu Simulator Game

A desktop game where you play as Matt Griffin, "fundraising" for a conference by managing overlapping menu windows, while dealing with Discord interruptions from Calvelli.

## Setup

1. Install Python 3.7 or higher
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Running the Game

```bash
python game.py
```

## Controls

- **Left Click**: Click and drag menu windows by their title bar
- **Left Click**: Click and drag items between menus
- **Tab**: Cycle through windows (Alt+Tab style)
- **ESC**: Exit the game

## Gameplay

- You have 5 overlapping menu windows to manage
- Move items between menus by dragging them
- Calvelli will interrupt you with Discord messages - close them to continue
- Watch the progress bar fill up as you "work"
- Complete the fundraising to see the ending!
